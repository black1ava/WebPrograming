Given a list of numbers and return the largest number

For the first sight, there are sereral solutions pop-up. Well, first of all an array can be declared to store the all the numbers. In order to find largest number, loop or qsort() build-in function can be used, otherwise you can create two functions larger and largest like this : 

        int larger(int a, int b){
          return a > b ? a : b;
        }
        
        int largest(int *array, int length){
          int max = INT_MIN;
          for(int i = 0; i < length; i++){
            max = larger(*(array + i), max);
          }
          return max;
        }

You can use Linked List to instead of array if you are good at data structure, since the problem said a list of numbers not an array of numbers.

By the way what is Time Complexity or Big-O-Notation?

Time Complexity

In computer science, the time complexity is the computational complexity that describes the amount of computer time it takes to run an algorithm.

There are a lot of type of time complexity, so today, we will talk about four types of time complexity :

  1. O(1)

  2. O(n)

  3. O(n2)

  4. O(log n);

Example in real life

Imagine a classroom of 100 students in which you gave your pen to one person. Now, you want that pen. Here are some ways to find the pen and what the O order is.

O(n2): You go and ask the first person of the class, if he has the pen. Also, you ask this person about other 99 people in the classroom if they have that pen and so on, This is what we call O(n2).

O(n): Going and asking each student individually is O(n).

O(log n): Now I divide the class into two groups, then ask: “Is it on the left side, or the right side of the classroom?” Then I take that group and divide it into two and ask again, and so on. Repeat the process till you are left with one student who has your pen. This is what you mean by O(log n).

I might need to do the O(n2) search if only one student knows on which student the pen is hidden. I’d use the O(n) if one student had the pen and only they knew it. I’d use the O(log n) search if all the students knew, but would only tell me if I guessed the right side.

Example in code : 

        printf("Hello World");
        
        Time Complexity : O(1)
      
        for(int i = 0; i < n; i++){
          printf("Hello world");
        } 
        
        Time Complexity : O(n)
      
        for(int i = 0; i < n; i++){
          for(int j = 0; j < n; j++){
            printf("Hello world");
          }
        } 
        
        Time Complexity : O(n2)
      
        for(int i = n; i >= 0; i /= 2){
          printf("Hello World");
        }
        
        Time Complexity : O(log n)
